software

Disclaimer: all those projects were developped for learning & fun. None of this is production ready, and I don't actively maintain any of them.

  • async-channels : Reimplementation of Clojure’s core.async channels in Java, including a Scala interface, to get functionalities similar to go channels.
  • CSharpRecords : A Visual Studio extension to auto-generate constructors & partial update method for immutable record classes.
  • js-regexp : A complete implementation of Javascript’s Regexp object in Javascript. My research intership project from a long time ago.
  • schemejvm : Somewhat silly meta-interpreter for a sub-set of Scheme, with its bootstrap interpreter written in Scala. Features JVM interop & Lisp style macro expansion.
  • gambit-oo : A Smalltalk inspired, minimal implementation of an object system for Gambit Scheme.
